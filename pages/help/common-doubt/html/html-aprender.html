o basico sobre html

O desenvolvimento front-end envolve criar a estrutura e a aparência dos sites. O primeiro passo para quem quer começar no front-end é aprender HTML, a linguagem básica para construir páginas web. A seguir, você encontrará uma trilha de aprendizado para ajudá-lo a dar os primeiros passos.

1. O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação utilizada para estruturar o conteúdo de uma página web. Ele permite adicionar elementos como textos, imagens, links e tabelas ao site. Para aprender o básico de HTML, acesse: Aprenda HTML Básico.

2. Estrutura Básica de uma Página HTML

Uma página HTML começa com uma estrutura simples. Aqui está um exemplo básico de uma página HTML: 

          
          <!DOCTYPE html>
          <html lang="pt-br">
            <head>
              <meta charset="UTF-8">
              <meta name="viewport" content="width=device-width, initial-scale=1.0">
              <title>Minha Primeira Página</title>
              <link rel="stylesheet" href="styles.css">
              <script src="script.js" defer></script>
            </head>
            <body>
              <h1>Olá, Mundo!</h1>
              <p>Este é o meu primeiro site com HTML.</p>
            </body>
          </html>

Esse código cria uma página com um título e um parágrafo. A seguir, vamos detalhar cada parte da estrutura da página HTML:

Declaração do Doctype

<!DOCTYPE html> é uma declaração que informa ao navegador que este documento está escrito em HTML5, a versão mais recente do HTML. Ela deve ser sempre a primeira linha do seu arquivo HTML.

Tag <html>

A tag <html> envolve todo o conteúdo da página e define que este é um documento HTML. O atributo lang="pt-br" define o idioma do conteúdo como português do Brasil, o que é importante para SEO e acessibilidade.

Seção <head>

A tag <head> contém metadados sobre o documento, que não são exibidos diretamente na página, mas ajudam o navegador e motores de busca a entender o conteúdo.

<meta charset="UTF-8">

O <meta charset="UTF-8"> especifica a codificação de caracteres do documento como UTF-8, que suporta a maioria dos caracteres de todas as línguas. Isso é essencial para garantir que caracteres especiais, como acentos, sejam exibidos corretamente.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

O <meta name="viewport" content="width=device-width, initial-scale=1.0"> é usado para definir o comportamento da página em dispositivos móveis. O atributo viewport garante que a largura da página seja ajustada de acordo com a largura do dispositivo. O valor width=device-width faz com que a página tenha a mesma largura da tela do dispositivo, enquanto initial-scale=1.0 define o nível de zoom inicial para 100%. Essa tag é essencial para o design responsivo.

<title>

A tag <title> define o título da página que aparecerá na aba do navegador e é importante para SEO. Neste caso, o título é "Minha Primeira Página".

Adicionando CSS

Para adicionar um arquivo CSS externo e estilizar sua página, utilizamos a tag <link>. O atributo rel="stylesheet" indica que este arquivo é uma folha de estilo e o atributo href aponta para o caminho do arquivo CSS. Exemplo: 

          
          <link rel="stylesheet" href="styles.css">

Isso vai carregar o arquivo styles.css que contém as regras de estilo que serão aplicadas à página.

Adicionando JavaScript

Para incluir um arquivo JavaScript externo, utilizamos a tag <script>. O atributo src define o caminho do arquivo JS, e o atributo defer garante que o JavaScript será executado somente após o carregamento completo da página HTML. Exemplo: 

          
          <script src="script.js" defer></script>

Isso carrega o arquivo script.js, que pode conter comportamentos interativos ou dinâmicos para sua página.

Seção <body>

A tag <body> contém todo o conteúdo visível da página, como textos, imagens, vídeos, links, etc. Tudo que é exibido no navegador para o usuário é colocado dentro da tag <body>.

Tags Importantes dentro do <body>

Algumas das principais tags que você vai utilizar para estruturar o conteúdo dentro da tag <body> são:

  • <h1> a <h6>: Tags de cabeçalhos, sendo <h1> o mais importante e <h6> o menos importante.
  • <p>: Define parágrafos de texto.
  • <a>: Cria links. Exemplo: Aprenda mais sobre links.
  • <img>: Insere imagens. Exemplo: Inserir imagens no HTML.
  • <ul> e <li>: Criam listas não ordenadas, onde cada item é definido por <li>.

Estrutura Completa

Agora você sabe como uma página HTML é estruturada e para que servem as tags principais. Para aprender mais sobre como combinar HTML com CSS e JavaScript, continue explorando os tutoriais seguintes:

  • Como adicionar e utilizar CSS
  • Como incluir e utilizar JavaScript

3. Elementos Fundamentais do HTML

No HTML, você usa "tags" para definir elementos como títulos, parágrafos, listas e imagens. Aqui estão alguns dos principais:

  • <h1> a <h6>: Tags de título, sendo <h1> o mais importante e <h6> o menor.
  • <p>: Usado para criar parágrafos de texto.
  • <a>: Tag usada para criar links. Exemplo: Aprenda sobre links.
  • <img>: Tag usada para inserir imagens. Exemplo: Inserir imagens no HTML.
  • <ul> e <li>: Usadas para criar listas não ordenadas (listas de itens).

Veja uma lista completa de elementos HTML em: Principais Elementos HTML.

4. Prática com Projetos Simples

A prática é essencial para aprender HTML. Comece criando pequenos projetos, como:

  • Uma página pessoal com suas informações básicas. Tutorial: Como criar uma página pessoal com HTML.
  • Um site simples com links para seus sites favoritos. Veja como fazer: Criar um site de favoritos com HTML.
  • Uma galeria de imagens. Saiba como: Criar uma galeria de imagens com HTML.

5. Hospedando seu Site no GitHub Pages com Hostinger

Depois de aprender o básico e criar seus primeiros projetos, você pode hospedar seu site gratuitamente no GitHub Pages. Veja os passos:

  1. Passo 1: Crie um repositório no GitHub e faça o upload dos arquivos HTML. Tutorial: Como criar um repositório no GitHub.
  2. Passo 2: Vá em Settings > Pages no GitHub e ative o GitHub Pages. Saiba mais: Como ativar o GitHub Pages.
  3. Passo 3: Configure seu domínio personalizado na Hostinger para apontar para o GitHub Pages. Tutorial: Configurar domínio Hostinger.

6. Continue Explorando

O HTML é a base de toda página web, mas ele não funciona sozinho. O próximo passo é aprender CSS para estilizar seu site. Veja mais sobre CSS em: Introdução ao CSS.

Com essa trilha, você terá uma boa base para começar no front-end. Continue praticando e explorando novas ferramentas!
